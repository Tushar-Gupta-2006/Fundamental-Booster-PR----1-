# Fundamental Booster (PR -- 1)

## Overview
A beginner-friendly Python notebook that demonstrates basic Python concepts such as variables, user input, data types, type conversion, arithmetic operations, and formatted output.

## Features
- Takes user input
- Displays data types
- Performs basic calculations
- Uses Python built-in functions
- Beginner-friendly examples

## Technologies
- Python 3
- Jupyter Notebook

## How to Run
1. Open the notebook in Jupyter Notebook.
2. Run all cells.
3. Enter the required inputs.
